# Codex CLI (distilled)

Source: https://developers.openai.com/codex/cli/

## Install
- `npm i -g @openai/codex`

## Run
- `codex` starts interactive TUI
- `codex exec "..."` runs non-interactively (scriptable)

## Notes
- Windows support is experimental; prefer WSL.
- CLI supports model selection, reasoning adjustments, image inputs, local code review, web search, cloud tasks, MCP, and approval modes.
